¿Cómo usar el Dashboard?
  • Configuración del Partido: Completa la fecha, hora, lugar, precio y observaciones
  • Guardar Equipos: Guarda automáticamente la configuración y los equipos en un solo paso
  • Flechas: Haz clic en las flechas para mover jugadores entre equipos
  • Movimiento Libre: Los equipos pueden quedar desbalanceados durante la edición
  • Generar Aleatoriamente: Crea nuevos equipos con la misma cantidad de jugadores
  • Guardar Equipos: Solo se guardan si ambos equipos tienen la misma cantidad de jugadores
  • Vista WhatsApp: Formato optimizado para captura de pantalla y compartir
¿Cómo funciona nuestro algoritmo de formación?
  • Balance por calificación: Los equipos se forman considerando las calificaciones de cada jugador.
  • Balance por edad: Los jugadores mayores se distribuyen equitativamente para evitar equipos de solo jóvenes vs solo mayores.
  • Distribución de arqueros: Si hay arqueros, se distribuyen equitativamente entre equipos.
  • Aleatoriedad: Cada regeneración crea equipos diferentes para mayor variedad.
  • Optimización inteligente: El sistema busca el mejor balance entre calificación, edad y distribución de arqueros.
  • Verificación de variedad: Se verifica que no se repitan equipos de partidos anteriores para mayor diversidad.
